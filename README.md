## Installation

packer:
```
use('jakebark/notes.nvim')
```
vim plug:
```
use('jakebark/notes.nvim')
```
lazy:
```
{
  'jakebark/notes.nvim',
  config = function() require("notes") end,
}
```
manual:
```
git clone https://github.com/jakebark/notes.nvim.git ~/.local/share/nvim/site/pack/notes/start/notes.nvim
```
